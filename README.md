# Weather Data Analysis Dashboard

## Project Description

This project analyzes historical weather data to understand temperature patterns, rainfall trends, and seasonal variations.

## Tools Used

* Microsoft Power BI
* Excel

## Dataset Fields

* Date
* City
* Temperature
* Humidity
* Rainfall
* WindSpeed
* WeatherCondition
* Season

## Dashboard Features

* Average Temperature KPI
* Temperature Trend Line Chart
* Rainfall Comparison by City
* Weather Condition Distribution
* Temperature vs Humidity Analysis

## Outcome

This dashboard helps in understanding climate trends and making data-driven decisions.
